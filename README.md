# GLV

Golang Latest Version

### Install

```
curl https://raw.githubusercontent.com/tourcoder/glv/master/glv | bash
```

### How to use

```
Usage: glv [OPTION]...
Options:
  -i, --install      Install/Update Golang
  -s, --setgopath    Set GOPATH
  -h, --help         Show help information
  -v, --version      Show GLV version
```
