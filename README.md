# GLV

Golang Latest Version

### Install

```
curl https://raw.githubusercontent.com/tourcoder/glv/master/glv -o /usr/local/bin/glv
chmod +x /usr/local/bin/glv
```

### How to use

Run command `sudo glv`, that's all.
